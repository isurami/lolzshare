# LolzShare

Минимальный файлообменник для просмотра медиа без кнопки скачивания.

## Возможности

- Регистрация и авторизация пользователей.
- Роли: обычный пользователь и админ. Админ может выдавать права на загрузку.
- Создание папок, загрузка фото/видео и управление видимостью.
- Доступ к папкам по ссылке или публично.

## Запуск

```bash
npm install
npm start
```

По умолчанию сервис стартует на `http://localhost:3000`.

Первый зарегистрированный пользователь автоматически получает права администратора и загрузчика.
